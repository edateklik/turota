# Recommendation Module

TasteProfile verisini Python FastAPI servisine gönderir; bölge, mekan, Timeline ve Explainable
AI sonuçlarını doğrulayıp PostgreSQL'e kaydeder. Domain/Application/Infrastructure projeleri
bu klasör altındadır. HTTP sözleşmesi için `docs/recommendation-integration.md` dosyasına bakın.
